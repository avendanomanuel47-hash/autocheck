Bienvenido a AutoCheck

Para guardar tus datos en la nube necesitás conectar un Google Sheet. Solo se hace una vez.

1
Creá el Google Sheet Abrí sheets.new en tu navegador y dale el nombre "AutoCheck DB"
2
Copiá el ID del Sheet En la URL: docs.google.com/spreadsheets/d/ESTE-ES-EL-ID/edit
3
Pegalo abajo y configurá También necesitás un Apps Script (instrucciones en el README incluido)

📲 Instalar AutoCheck
Seguí estos pasos en Chrome para tener el ícono en tu pantalla de inicio:
1
Tocá los 3 puntos (⋮) arriba a la derecha de Chrome
2
Elegí "Agregar a pantalla de inicio" o "Instalar app"
3
Confirmá tocando "Agregar" o "Instalar"
💡 Si abriste este archivo desde WhatsApp o Drive, primero tocá los 3 puntos del visor y elegí "Abrir en Chrome" — el menú de instalación solo aparece dentro de Chrome.
